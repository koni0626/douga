# 第三者ソフトウェアと商用利用

本アプリのコードとは別に、配布・ホスティング時に各依存物のライセンスと利用規約を確認する。

- FFmpeg はビルド構成で LGPL/GPL 条件が変わる。本開発環境は `--enable-gpl` と libx264 を使うため GPL 構成として扱い、サーバー側の独立プロセスとして実行する。配布する場合は対応するソース入手方法、ライセンス表示、改変内容を法務確認する。
- H.264 の特許・配信条件はソフトウェアライセンスとは別に、配信地域と用途に応じて確認する。
- OpenAI API の入力データ、生成物、利用制限、プライバシー説明は公開時点の契約・ポリシーに合わせる。
- AivisSpeech EngineはDougaとは別プロセスとして利用する。Engine本体はLGPL v3と別ライセンスのデュアルライセンスで提供されているため、配布形態に応じて[公式リポジトリ](https://github.com/Aivis-Project/AivisSpeech-Engine)の条件を確認する。音声モデルはモデルごとに利用規約・クレジット条件が異なるため、公開前に使用モデル単位で確認する。
- npm/Python/container の依存ライセンス一覧を各リリースで生成し、禁止ライセンスと既知脆弱性を CI で検査する。

これは法的助言ではない。商用公開前に対象地域の専門家へ確認する。
