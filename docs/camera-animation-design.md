# カメラアニメーション設計

## 1. 目的

キャンバス全体へ位置・回転・拡大率のループ変換を適用し、静止画中心の動画に手持ち撮影、一人称視点、呼吸、浮遊などのカメラワークを加える。プレビューとMP4出力は`scene-renderer`の同一計算を利用する。

## 2. UI

- 編集ツールに「カメラ」を設ける。
- タイムライン最上段にカメラ専用トラックを常時表示する。
- プリセット追加後、開始・終了、強さ、1周期の長さを編集できる。
- カメラエフェクトの終了が動画時間を超える場合は、動画時間を50ミリ秒単位で自動延長する。

初期プリセット:

- 微細な手ブレ
- 歩行
- 呼吸
- 浮遊
- 左右スイング
- ゆっくり回転
- ズーム往復
- 心拍

## 3. データモデル

プロジェクト直下の`camera_effects`へ、ID、プリセット、開始・終了時刻、強さ、周期を保存する。複数エフェクトが同じ時刻に有効な場合、移動量・回転角度・拡大量を合成する。

## 4. 描画

背景、画像、テキスト、図形、テロップを含むシーン全体を、キャンバス中央基準で移動・回転・拡大する。編集用の選択枠にも同じ変換と逆座標計算を適用し、カメラ移動中も操作位置を一致させる。
